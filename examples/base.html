安装与使用

一、安装方法

npm install virtualscroll --save

二、使用方法

构建方法一，页面引入
<script src="node_modules/VirtualScroll/build/VirtualScroll.js"></script>
VirtualScroll(DOM对象);
构建方法二，require
let VS = require('VirtualScroll');
VS(DOM对象);
构建方法三, import
import VS from 'VirtualScroll';
VS(DOM对象);
